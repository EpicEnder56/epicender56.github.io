Linux

I do not use Windows on my main PC. Instead, I use Linux: a free and open source alternative. My go to distro is Linux Mint as it combines my favourite distro, Ubuntu, with my favourite Desktop environment, Cinnamon. On older machines, my distro of choice is Lubuntu as this runs well on older hardware and as a modern interface. When using my Raspberry Pi 4, I use Ubuntu Server as a base, adding Cinnamon desktop on top. For my Raspberry Pi 3B and 3B+, I use Raspberry Pi OS lite as a base, then add the mate desktop.
