Raspberry Pi 4

The Raspberry Pi 4 is the latest and greatest Raspberry Pi. It comes with 1, 2, 4 or 8 gigabytes of RAM. It has the Broadcom BCM2711 SoC (System on a chip) with the quad-core Cortex-A72 processor running at 1.5GHz. Its GPU is the Broadcom VideoCore VI. It is the second generation of Pi to support ARM64 architectures, as well as the older ARMhf architecure used on the Pi 2, 1 and 0.

Operating systems

All official OSes can be downloaded from the Raspberry Pi download site. My personal favourite is Ubuntu server, but for beginners I'd recommend Raspberry Pi OS as this is easy to setup on any micro SD card using the Raspberry Pi imager.
